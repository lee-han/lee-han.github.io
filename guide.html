조선시대 배치도 설명

※ 일러두기

  • 이 사이트는 조선시대 각종 배치도에 나온 명칭을 손쉽게 찾기 위해 만들어습니다.
  • 이 사이트에서 사용한 자료는 인터넷상에서 구한 것입니다. 출처가 분명한 경우에는 명시했으나, 그렇지 않은 경우에는 적을 수 없었습니다. 혹여나 저작권 협의가 필요하다면 연락해주세요.
  • 모든 결과물은 개개인의 기여로 만들었으며, 배치도마다 기여자를 명시하였습니다.

※ 데이터 로딩 에러 조치

  • 왼쪽 패널 아래쪽에 데이터가 없으면 '데이터 로딩 에러 시’ 아래 링크를 클릭한 후 영어로 Request...라고 적혀있는 버튼을 누릅니다.
  • 그러면 버튼 아래에 영어로 You currently have temporary access...라고 나옵니다. 영어 창을 닫아주세요.
  • 배치도가 나오는 원래 창을 새로고침 하면 데이터가 정상적으로 로딩됩니다.
  • 로딩 에러가 뜰 때마다 이렇게 조치해주세요.

※ 데이터 수정 요청

  • 명칭이나 위치가 틀린 곳을 발견하면 알려주세요.
  • 오른쪽 오류 신고를 클릭하시고 배치도, 명칭 등을 적어주시면 수정하겠습니다. 오류 신고

데이터 입력 방법

  • 배치도의 데이터는 ‘명칭, 한자, 위치, 분류, 이미지’ 총 5가지 항목으로 구성되어 있습니다. 수선전도의 경복궁은 경복궁 景福宮 1275, 615 건물 수선전도라고 되어 있습니다.
  • 이미지만으로 판독이 어려운 경우에는 경★궁처럼 별표로 입력하면 추후에 확인하여 수정하겠습니다. 별표는 한글로 'ㅁ' 입력 후 한자키를 누르면 나옵니다.
  • 데이터 입력은 구글 설문을 사용합니다. 5개 항목을 작성하여 제출하면 약 10초 후에 데이터가 반영됩니다. (새로고침 필요)
  • 위치를 마우스로 찍을 때는 명칭 바로 위쪽을 찍어주세요.
  • 데이터 저장은 구글 스프레드시트를 사용합니다. 링크

명칭

  • 한글 명칭을 입력합니다.
  • 필요하다면 검색을 위하여 별도 명칭을 병기해도 무방합니다. 정(우물), 이산(니산)

한자

  • 한자를 입력합니다. 꼭 필요한 경우 외에는 굳이 이체자로 입력하지 말고 정자로 입력합니다.

위치, 이미지

  • 오른쪽 패널의 이미지에서 특정 지점을 클릭하면 그 지점의 위치가 왼쪽 패널에 기록됩니다.
  • 위치가 기록된 후에 데이터 입력 버튼을 누르면 구글 설문이 열립니다.
  • 위치와 이미지는 자동으로 입력되어 있습니다.

분류

  • 각 명칭이 문(門)인지 건물인지 지명인지 혹은 기타인지 입력합니다.
  • 건물인지 지명인지 헷갈리면 미상을 선택해주세요. 추후에 확인하여 수정하겠습니다.
  • 3가지에 속하지 않는다면 기타를 클릭하고 분류명을 입력해주세요.
  • 문은 파란색으로 표시되고, 건물은 빨간색으로 표시되며, 지명은 녹색으로 표시됩니다. 기타는 노란색으로 표시됩니다.

기여자

  • 기여자는 왼쪽 패널의 제목 아래에 표시됩니다.
  • 실명, 호, 별명, ID 등 아무거나 상관없습니다.
  • 데이터 건마다 적을 필요 없이 아무때나 1번만 적으시면 됩니다. 같은 기여자를 여러 번 적더라도 1번만 표시됩니다.

동궐도 추가 지침

  • 동궐도는 크기가 너무 커서 가로 16개, 세로 6개의 표 형태로 잘라놓았습니다.
  • 왼쪽 패널의 고화질 버튼을 누르면 오른쪽 패널에 흰색 표와 숫자가 나옵니다. (101호부터 116호까지, 총 6층인 아파트라고 상상하시면 눈에 잘 들어옵니다.)
  • 입력하고자 하는 칸을 클릭하면 새로운 탭으로 고화질 이미지가 나옵니다. 전체 이미지와 고화질 이미지가 정확하게 맞지는 않습니다. 혹시 짤려서 보이지 않는다면 이어지는 부분을 확인해주세요.
  • 고화질 이미지를 확대하여 명칭을 확인한 후에 고화질 버튼을 한번 더 눌러서 아파트같은 표를 끕니다.
  • 이미지의 해당 부분을 클릭해서 위치값이 왼쪽 패널에 찍히면 데이터 입력 버튼을 눌러 위에 있는 가이드대로 3가지 항목을 입력한 후 구글 설문을 제출합니다.
